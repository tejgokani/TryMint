## Quickstart (quickstart.md)

Steps:
1. Clone repository
2. Install dependencies
3. Configure environment
4. Start backend
5. Start frontend
6. Install agent
7. Connect and test

## Prerequisites (prerequisites.md)

Requirements:
- Node.js >= 18.x
- npm >= 9.x
- Git
- Google OAuth credentials
- (Optional) Redis
